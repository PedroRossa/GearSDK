# GearSDK
An openSource project to facilitate the use of the NodeMCU with diffrent languages.

Create objects on SDK with your sensors.

#Wrappers
 - C++
 - C#
 
 
 #Examples
  - C++ Console
  - C++ with SFML
  
  - C# Console
  - C# Windows Forms
  - C# Unity 3D

You want to contribute, make a fork, update something and make a pull request :D
